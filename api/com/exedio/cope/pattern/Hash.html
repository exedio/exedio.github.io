        public final Hash limit(int plainTextLimit)
        Creates a new hash with a new plain text limit. The default is 150. Setting hashes longer than the limit will cause an Hash.InvalidPlainTextException. Checking for hashes longer than the limit will silently return false. This is a precaution against DOS attacks with very long plain texts.
