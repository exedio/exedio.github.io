com.exedio.cope.misc

Annotation Type Computed


  • @Target(value={TYPE,FIELD})
@Retention(value=RUNTIME)
@Inherited
public @interface Computed
    This annotation marks a feature or type as computed. This means, that the persistent contents of that feature or type are typically not modified by human users, but by some computational process. So any generic editing facility is advised to not let a human user modify this type or feature.

    For types, this means, that items of the type shall not be created or deleted by human users. For features this means, that the state of the feature shall not be modified by a human user. A computed type does not imply, that the type's features are computed as well.

    When creating items, a computed feature must not be specified via a SetValue given to Type.newItem, so that any default mechanisms such as defaultTo can take effect. As a consequence, items of types with computed features, that are mandatory and have no default mechanisms, cannot be created.

    This annotation does not specify, whether this feature or type shall be shown to a human user.
